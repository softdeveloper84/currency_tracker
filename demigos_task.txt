Тестовое задание на Middle Python Developer

Необходимо сохранять в БД курс валют BTC-USD с интервалом в 10с.
Сервис должен иметь веб интерфейс - сайт, где можно увидеть текущее значение курса.
Пользователь может ввести новую пару валют, и она добавится к списку на получение.

Курс доступен на многих сайтах, например, тут https://ru.cryptonator.com/rates/BTC-USD 

Желательно использовать Django.

Код заливайте в репозиторий Github или Bitbucket.
